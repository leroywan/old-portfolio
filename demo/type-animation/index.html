Type-Animation.js

Simple Animated Typing For Your Websites

Created by Leroy Wan

Demo

Demo #1 - Simple Lazy Code (No CSS or Javscript Needed!)

All you need to do is add a class name of "type-animation" to your tags and watch the magic happen.

Input:

<p class="type-animation"> This is the first demo. I am executed once the page loads! </p>

Output:

This is the first demo. I am executed once the page loads!

*Note that this is animation begins right after the page loads. Restart (F5) the page to see the animation again.

Demo #2 - Blinking Text Cursor Without CSS?

Add a class name of "enable-cursor" to your "type-animation" tags and you are good to go!

Input:

<p class="type-animation enable-cursor"> This is the second demo. I come with a cursor enabled! </p>

Output:

This is the second demo. I come with a cursor enabled!

*Note that this is animation begins right after the page loads. Restart (F5) the page to see the animation again.

Demo #3 - Lets Race to See Who's Faster!

Add the class names "type-animation-slow" or "type-animation-fast" to control the speed of your animation

Input:

<p class="type-animation-slow enable-cursor"> This is the third demo. I feel like I am kinda slow at this. </p>

<p class="type-animation-fast enable-cursor"> This is the third demo. I am totally gonna win this race thing! </p>

Output:

This is the third demo. I feel like I am kinda slow at this.

This is the third demo. I am totally gonna win this race thing!

*Note that this is animation begins right after the page loads. Restart (F5) the page to see the animation again.

Demo #4 - Did I Hear an Event Triggered?

This is where the JavaScript kicks in. In your HTML file, add the class "type-animation-trigger-once" to your tag and see what happens!

Input:

<p id="demo-4-content" class="type-animation-trigger-once"> This is the fourth demo. I'll show up whenever you want me to show up! </p>

Output:

This is the fourth demo. I'll show up whenever you want me to show up!

Nothing will show up until this button is pressed PRESS ME!

Javascript:


var demo4Button = document.getElementById('demo-4-button');
var demo4Content = document.getElementById('demo-4-content');

//start animation of element with the ID demo4Content once click event is triggered demo4Button.addEventListener('click', function(){
  demo4Content.typeAnimation.play();
})

Usage: element.typeAnimation.play()

The class "type-animation-trigger-once" assigns the object typeAnimation to your element which contains the play() method. Upon a trigger of an event such as a button click or scrolling, the element will be animated.

*Note that this animation can only be triggered once. Use class="type-animation-trigger" for repeated animation

Demo #5 - Repeat, Repeat, Faster!

Using the JavaScript rule as demo #4, add the class "type-animation-trigger" to repeat your animation instead! Also, the play() method by default has a speed set to 60. The lower the number the faster the animation runs!

Usage: element.typeAnimation.play(speed)

Input:


<p id="demo-5-content" class="type-animation-trigger">
  This is the fifth demo. I can repeat and I am faster than normal!
</p>

Javascript:


//start animation of element with the ID demo5Content once click event is triggered demo5Button.addEventListener('click', function(){
  demo5Content.typeAnimation.play(15);
})

Output:

This is the fifth demo. I can repeat and I am faster than normal!

Nothing will show up until this button is pressed PRESS ME!

Demo #6 - Lets Start Linking Things Together!

Add the class "type-animation-trigger" to your element and you can manipulate what occurs after the animation reaches completion!

Usage: element.typeAnimation.play(speed, callback)

Input:


<p id="demo-6-content" class="type-animation-trigger">
  This is the sixth demo. I can start being creative with my callback functions!
</p>

Javascript:


//start animation of element with the ID demo6Content once click event is triggered demo6Button.addEventListener('click', function(){
  demo6Content.typeAnimation.play(15, function(){
    alert('It works!')
    //Hint: You can link animations together to have them play one after the other

    anotherElement.typeAnimation.play(50) <-runs after completion
  });
})

Output:

This is the sixth demo. I can start being creative with my callback functions!

The callback (with the alert) will run when the animation ends PRESS ME!

Demo #7 - Special Characters?

The characters ` and @ gives you the ability to indent and create linebreaks. The characters can be escaped using an \

Usage: @ = linebreak, ` = 5 spaces

Input:


<p id="demo-7-content" class="type-animation-trigger enable-cursor">
This is the seventh demo. Here is 1 line break@Here are 2 linebreaks@@And now a linebreak with an indentation!@`You can imitate writing code like this!@@Example:@@function(){@`alert('this is awesome!')@};@@Escape character example: type-animation\@example.com
</p>

Output:

This is the seventh demo. Here is 1 line break@Here are 2 linebreaks@@And now a linebreak with an indentation!@`Imitate writing code like this!@@Example:@@function(){@`alert('this is awesome!')@};@@Escape character example: type-animation\@example.com
